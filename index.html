CSS Selector reference sheet

Name Example Meaning
UNIVERSAL * {} Applies to all elements in the document
Targets all elements on the page
TYPE h1, h2, h3 {} Matches element names
Targets the <h1>, <h2> and <h3> elements
CLASS .note {} Matches an element whose class attribute has a value that matches the one specified after the period (or full stop) symbol
Targets any element whose class attribute has a value of note
p.note {}
Targets only <p> elements whose class attribute has a value of note
ID #introduction {} Matches an element whose id attribute has a value that matches the one specified after the pound or hash symbol
Targets the element whose id attribute has a value of introduction
CHILD li>a {} Matches an element that is a direct child of another
Targets any <a> elements that are children of an <li> element (but not other <a> elements in the page)
DESCENDANT p a {} Matches an element that is a descendent of another specified element (not just a direct child of that element)
Targets any <a> elements that sit inside a <p> element, even if there are other elements nested between them
